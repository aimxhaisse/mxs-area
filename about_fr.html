My name is Sébastien Rannou, I'm a french student interested in computer security, programming languages (C, C++, lisp, JavaScript, PHP), and unix systems. I'm currently an exchange student at DSV (Stockholm).

About this place

I've had several blogs before, and I've been switching from one engine to another until I decided to make my own one in lisp. I've never used lisp for the web before, so this is a place where I usually experiment stuff.

My projects

I've been working on the following projects (from the most recent to the oldest):

  • des: an implementation of DES (Data Encryption Standard)

  • c0depast3r: the best pastebin tool ever hosted on buffout
  • aimxhOS: a little kernel which does... nothing
  • dofuspic : an image generator for characters in Dofus

I've also been working on several scholar projects such as a shell, a modular webserver or a raytracer.

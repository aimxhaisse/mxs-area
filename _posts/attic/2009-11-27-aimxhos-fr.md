## Description

Un petit kernel qui ne fait... rien :-) ; plus un endroit
pour expérimenter plutôt qu'un projet sérieux, néanmoins les
points suivants sont implémentés :

 * timer
 * gdt/idt/isr
 * vga/keyboard
 * scheduler
